Lumina is a university announcements and events awareness app that lets users view campus updates, interact with posts, and receive notifications. Access and available screens vary by role (Student, Staff, Admin).
Purpose of the application
  •	Keep students informed about announcements and events
  •	Help staff publish announcements/events and monitor engagement
  •	Help admins manage users and academic structures (campus/college/course) and view activity/analytics
Key functionality
  •	Announcement & event feed with search, filtering, and sorting
  •	Post details with images, likes, comments, and replies
  •	Event RSVP (“Going”), participant counts, and participant list viewing
  •	In-app notifications + push notification support (configurable)
  •	Role-based dashboards (admin/staff) with statistics and charts
  •	Admin management modules (students, staff, campus, college, course, activity logs)
